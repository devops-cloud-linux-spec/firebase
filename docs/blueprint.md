# **App Name**: Momentum Tracker

## Core Features:

- User Authentication: Allow users to create an account, log in securely, and manage basic profile settings to personalize their habit tracking journey.
- Habit Creation & Management: Enable users to define new daily habits (e.g., reading, workout) with customizable names, targets, and categories.
- Daily Habit Tracking: Provide a simple and intuitive interface for users to quickly mark their habits as completed or missed each day.
- Streak & Progress Visualization: Clearly display current habit streaks and show historical completion data through an interactive chart or calendar view to motivate consistency.
- AI-Generated Motivation: Utilize a generative AI tool to provide dynamic and personalized motivational messages or prompts based on user progress or detected habit patterns, offering positive reinforcement.
- Customizable Habit Reminders: Allow users to set specific times for daily in-app notifications or push reminders to ensure they don't forget to complete their habits.

## Style Guidelines:

- Primary color: A rich, calm green (#4D7A1F) symbolizing growth and focus, providing strong contrast on a light background.
- Background color: A very light, desaturated green (#F2F5F0) creating a clean and subtle base that encourages concentration.
- Accent color: A vibrant, energetic yellow-green (#A6F25A) for highlights, interactive elements, and positive feedback.
- Headline and body text font: 'Inter', a grotesque-style sans-serif for its modern, neutral, and highly readable appearance across all text.
- Employ a consistent set of minimalist line-art style icons that clearly represent habit actions, progress, and settings, maintaining a clean and modern aesthetic.
- Implement a dashboard-style layout prioritizing clear visibility of daily habits and immediate progress. Utilize clean, card-based components for individual habits and responsive design for various screen sizes.
- Incorporate subtle animations for user interactions, such as smooth transitions between screens, and micro-animations for positive feedback (e.g., marking a habit complete) to enhance user engagement without distraction.
